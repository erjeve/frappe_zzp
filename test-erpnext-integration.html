ERPNext File Integration Test

Manual Testing Steps:

1. Configure N8N Webhook

• Replace trigger with Webhook node

• Set path: erpnext-file-upload

• Your webhook URL: https://n8n.fivi.eu/webhook/erpnext-file-upload

2. Create ERPNext Webhook

• Go to: Setup → Integrations → Webhook

• Document Type: File

• Event: After Insert

• Request URL: https://n8n.fivi.eu/webhook/erpnext-file-upload

3. Test File Upload

• Create a new Purchase Invoice in ERPNext

• Attach a PDF invoice file

• Check N8N workflow executions
